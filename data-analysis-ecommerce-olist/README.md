# 🛒 Data Analysis – E-commerce Olist

## Overview
Conducted an in-depth exploratory data analysis (EDA) on the Brazilian Olist e-commerce dataset to uncover patterns in customer behaviour, sales performance, and logistics efficiency.

## Tools & Technologies
- **Python** – data wrangling and analysis
- **Pandas & NumPy** – data manipulation and aggregation
- **Matplotlib & Seaborn** – data visualisation
- **Jupyter Notebook** – interactive analysis environment

## Key Analysis Areas
- Customer segmentation and purchasing patterns
- Sales trends by product category, region, and time period
- Delivery performance and its correlation with customer satisfaction (review scores)
- Revenue distribution and top-performing sellers
- Payment method preferences and order value analysis

## Dataset
- Source: Olist Brazilian E-commerce Public Dataset
- Scope: 100,000+ orders across multiple product categories
- Features: order status, price, freight, delivery times, customer reviews
